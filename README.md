## Project Description:
This is a personal portfolio website built using HTML and CSS.
It includes:

- Header with name and tagline
- Navigation bar
- Home section (Hobby with ordered list)
- About section
- Projects section (3 projects)
- Contact form
- Footer with LinkedIn and GitHub links
- Responsive design using Flexbox

## How to Run the Project:

1. Download the folder.
2. Open index.html in any web browser (Chrome, Edge, Firefox).
3. No installation required.
